Frequently asked questions

What is FactCheck?

FactCheck is an algorithm that validates facts by finding evidences on the web to approve or deny a query. facts can be entered using 'Text' or 'File' input.

 How to use?

Once the fact is submitted, the API returns the total score as well as proof sentences and their corresponding scores. These features are described below.

  • Text Input:
    In 'Text' input. Subject and Object labels must be provided. You must put atleast one label in subject and object input box and hit Enter key. You must have to enter at least one subject and one Object label and select a predicate from the drop down list. Subject URI and Object URI is optional.

    Example:
    Albert Einstein awarded Nobel Prize in Physics.     In this example, Albert Einstein is subject , award is predicate and Nobel Prize in Physics is object.

  • File Input:
    In the File input, only .ttl file must be provided.
How to enter multiple labels at once?

Multiple labels for subject or object can be entered using a ',' (comma) between them.


Example:
Albert Einstein, Albert, Albert Einst
typing lables above and hitting enter will inserts three labels at once in subject or object input. Example ttl file
